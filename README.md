# hive-udfs

The repository is a collection of UDFs for Hive.

## Requirements
- Hive
- Java JDK (requested to build)
- Gradle 4.4.1 or later (requested to build)

## Build
1. clone this repository.
1. move to repository root

    ```
    cd hive-udfs
    ```

1. run to build

    ```
    gradle build
    ```
